# Система премиум-постов с медиа

## Обзор

Система премиум-постов позволяет пользователям создавать объявления с фото или видео за €50. Посты публикуются только после подтверждения оплаты администратором.

## Для пользователей

### Как создать премиум-пост:

1. **Выберите "💎 Премиум-пост с медиа"** в главном меню
2. **Загрузите медиа** - фото или видео для объявления
3. **Заполните описание** - минимум 10 символов, без ссылок и контактов
4. **Укажите социальные сети** - или напишите "нет"
5. **Введите Telegram @username**
6. **Укажите основной телефон** - формат +35191xxxxxxx, +35192xxxxxxx, +35193xxxxxxx или +35196xxxxxxx
7. **Укажите WhatsApp** - если отличается от основного телефона
8. **Введите имя или название компании**
9. **Подтвердите создание** - просмотрите предварительный просмотр
10. **Оплатите €50** - следуйте инструкциям
11. **Ожидайте подтверждения** - администратор проверит оплату

### Что включено в премиум-пост:

- ✅ Публикация с медиа в канале
- ✅ Приоритетное размещение
- ✅ Увеличенная видимость
- ✅ Профессиональное оформление

### Статусы постов:

- **pending** - ожидает подтверждения оплаты
- **approved** - оплата подтверждена, готов к публикации
- **rejected** - оплата отклонена

## Для администраторов

### Команды управления:

#### `/premium_posts`
Показывает список ожидающих подтверждения премиум-постов.

**Пример:**
```
/premium_posts
```

#### `/approve_payment <post_id> [заметки]`
Подтверждает оплату премиум-поста.

**Примеры:**
```
/approve_payment 123
/approve_payment 123 Оплата получена
```

#### `/reject_payment <post_id> <причина>`
Отклоняет оплату премиум-поста.

**Примеры:**
```
/reject_payment 123 Оплата не поступила
/reject_payment 123 Неверная сумма
```

### Процесс подтверждения:

1. **Проверьте список** - `/premium_posts`
2. **Проверьте оплату** - получите подтверждение от пользователя
3. **Подтвердите или отклоните** - используйте соответствующие команды
4. **При подтверждении** - пост готов к публикации в канале

## Структура базы данных

### Таблица `premium_posts`:
- `id` - уникальный ID поста
- `user_id` - ID пользователя
- `mode` - тип поста (seeking/offering)
- `cities` - города
- `description` - описание
- `social_media` - социальные сети
- `telegram_username` - Telegram username
- `phone_main` - основной телефон
- `phone_whatsapp` - WhatsApp
- `name` - имя/название компании
- `media_file_id` - ID медиа файла
- `media_type` - тип медиа (photo/video)
- `payment_status` - статус оплаты
- `payment_amount` - сумма оплаты (€50)
- `admin_notes` - заметки администратора
- `message_id` - ID сообщения в канале
- `chat_id` - ID чата
- `topic_id` - ID топика
- `status` - статус поста
- `created_at` - дата создания
- `updated_at` - дата обновления

### Таблица `payments`:
- `id` - уникальный ID платежа
- `premium_post_id` - ID премиум-поста
- `amount` - сумма платежа
- `payment_method` - способ оплаты
- `payment_proof` - подтверждение оплаты
- `status` - статус платежа
- `approved_by` - ID администратора
- `approved_at` - дата подтверждения
- `admin_notes` - заметки администратора
- `created_at` - дата создания

## Безопасность

- Только администраторы могут подтверждать/отклонять оплаты
- Все действия логируются
- Медиа файлы сохраняются в Telegram
- Проверка валидности всех данных

## Монетизация

- **Стоимость:** €50 за премиум-пост
- **Включено:** Публикация с медиа, приоритетное размещение
- **Процесс:** Предоплата → Подтверждение → Публикация
