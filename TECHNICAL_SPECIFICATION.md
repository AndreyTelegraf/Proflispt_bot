# Техническое задание: Telegram Bot "Work in Portugal"

## 📋 Общее описание

**Название**: Work in Portugal Bot  
**Username**: @workinportugal_bot  
**Назначение**: Бот для публикации объявлений о поиске и предложении работы в Португалии

## 🎯 Основные функции

### 1. Публикация объявлений о поиске работы
- Сбор информации о соискателе
- Форматирование объявления по стандарту
- Публикация в соответствующий топик канала

### 2. Публикация объявлений о предложении работы
- Сбор информации о вакансии
- Форматирование объявления по стандарту
- Публикация в соответствующий топик канала

### 3. Управление объявлениями
- Редактирование существующих объявлений
- Удаление объявлений
- Просмотр своих объявлений

## 🔄 Пользовательский сценарий

### Сценарий 1: Публикация объявления о поиске работы

1. **Старт**: Пользователь отправляет `/start`
2. **Выбор типа**: Пользователь выбирает "Ищу работу"
3. **Выбор города**: Пользователь выбирает город(а) или "Онлайн"
4. **Описание**: Пользователь описывает искомую работу
5. **Instagram**: Пользователь указывает Instagram или "нет"
6. **Telegram**: Пользователь указывает свой @username
7. **Телефон**: Пользователь указывает основной номер (+351...)
8. **WhatsApp**: Пользователь указывает WhatsApp или "тот же"
9. **Имя**: Пользователь указывает ФИО
10. **Предпросмотр**: Бот показывает форматированное объявление
11. **Публикация**: Пользователь подтверждает публикацию

### Сценарий 2: Публикация объявления о предложении работы

1. **Старт**: Пользователь отправляет `/start`
2. **Выбор типа**: Пользователь выбирает "Предлагаю работу"
3. **Выбор города**: Пользователь выбирает город(а)
4. **Описание**: Пользователь описывает вакансию
5. **Instagram**: Пользователь указывает Instagram компании или "нет"
6. **Telegram**: Пользователь указывает @username компании
7. **Телефон**: Пользователь указывает контактный номер
8. **WhatsApp**: Пользователь указывает WhatsApp или "тот же"
9. **Название**: Пользователь указывает название компании
10. **Предпросмотр**: Бот показывает форматированное объявление
11. **Публикация**: Пользователь подтверждает публикацию

## 🗄️ База данных

### Таблица users
```sql
CREATE TABLE users (
    id INTEGER PRIMARY KEY,
    telegram_id INTEGER UNIQUE NOT NULL,
    username TEXT,
    first_name TEXT,
    last_name TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### Таблица job_postings
```sql
CREATE TABLE job_postings (
    id INTEGER PRIMARY KEY,
    user_id INTEGER NOT NULL,
    mode TEXT NOT NULL, -- 'seeking' or 'offering'
    cities TEXT NOT NULL, -- JSON array
    description TEXT NOT NULL,
    instagram TEXT,
    telegram_username TEXT NOT NULL,
    phone_main TEXT NOT NULL,
    phone_whatsapp TEXT,
    name TEXT NOT NULL,
    message_id INTEGER, -- ID сообщения в канале
    chat_id INTEGER, -- ID канала
    topic_id INTEGER, -- ID топика
    status TEXT DEFAULT 'active', -- 'active', 'deleted', 'expired'
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users (id)
);
```

### Таблица drafts
```sql
CREATE TABLE drafts (
    id INTEGER PRIMARY KEY,
    user_id INTEGER NOT NULL,
    mode TEXT,
    cities TEXT,
    description TEXT,
    instagram TEXT,
    telegram_username TEXT,
    phone_main TEXT,
    phone_whatsapp TEXT,
    name TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users (id)
);
```

### Валидация данных
