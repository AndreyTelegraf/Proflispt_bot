# Инструкция по бану пользователей

## Получение Telegram ID пользователя

К сожалению, получить ID пользователя по username (@username) через Bot API невозможно без взаимодействия пользователя с ботом.

### Способы получения ID:

1. **Через пересланное сообщение** (самый простой способ):
   - Попросите пользователя написать боту любое сообщение
   - Или перешлите сообщение от пользователя боту
   - Используйте команду `/getid` в ответ на пересланное сообщение

2. **Через бота @userinfobot**:
   - Напишите боту @userinfobot
   - Отправьте ему username пользователя или перешлите сообщение
   - Бот вернет ID пользователя

3. **Через веб-версию Telegram**:
   - Откройте профиль пользователя в веб-версии Telegram
   - ID можно найти в URL или через инструменты разработчика

## Бан пользователей

### Через команду в боте:
```
/ban <telegram_id> Причина
/ban <telegram_id> Причина 7d  (временный бан на 7 дней)
/ban <telegram_id> Причина 24h (временный бан на 24 часа)
/ban @username Причина  (только если пользователь уже взаимодействовал с ботом)
```

### Через скрипт:
```bash
python3 ban_users.py <telegram_id1> [telegram_id2] ... [причина]
```

Пример:
```bash
python3 ban_users.py 123456789 987654321 Нарушение правил
```
